# GEMINI.md

このファイルは Gemini CLI に対する専用の指示書です。

## 共通ガイドライン
プロジェクトのルール、アーキテクチャ、コミットコンベンションについては **[RULES.md](./RULES.md)** を参照してください。

## Gemini CLI 特定ルール
- Windows (win32) 環境で実行されているため、PowerShell コマンドと WSL2 (Ubuntu) 上のコマンドを明確に区別して提案してください。
- ファイル操作を行う際は、`.gitignore` による広範な除外設定を考慮し、必要に応じて管理対象の確認を行ってください。
- 新しいツールの追加や設定変更の際は、`.config/mise/config.toml` への反映を検討してください。
